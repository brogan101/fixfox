# Support

## Before Contacting Support
1. Reproduce the issue once.
2. Run the relevant Home or IT runbook.
3. Create a `ticket` support bundle from Reports.
4. Copy ticket summary (short + detailed).

## What to Include
- FixFox version (About page)
- Symptom and exact repro steps
- Export ZIP path or attached ZIP
- Any visible error text from ToolRunner

## Support Bundle Contents
- App version, channel, Python/runtime platform, and bundle preset metadata.
- Session summary, findings count, actions count, and evidence count.
- Safe settings snapshot with layout-only values removed.
- Recent application log and crash log when bundle logging is enabled.
- The validated report, raw session JSON, diagnostics log, ticket summaries, and collected evidence.

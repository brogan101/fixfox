# Support

## Before Contacting Support
1. Reproduce the issue once.
2. Run the relevant Home or IT runbook.
3. Export a `ticket` support pack from Reports.
4. Copy ticket summary (short + detailed).

## What to Include
- Fix Fox version (About page)
- Symptom and exact repro steps
- Export ZIP path or attached ZIP
- Any visible error text from ToolRunner

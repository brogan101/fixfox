# FixFox Quick Start

FixFox is a Windows support app for three jobs:

1. Check what needs attention.
2. Run safe, explainable fixes.
3. Package the issue cleanly when self-service is not enough.

## Start here

- Open **Home** to see the next safe action.
- Run **Quick Scan** to surface the current health picture.
- Use **Guided Diagnosis** when the problem is easier to describe than classify.
- Use **Repair Library** when you already know the repair you need.
- Use **Automation** for maintenance profiles and repeatable workflows.
- Use **Support Package** when the issue still needs escalation.

## What the tray does

- FixFox can stay in the system tray when closed if you enable that in Settings.
- The tray menu can reopen the app, run a safe maintenance workflow, run a quick scan, or create a support package.
- If work is still running, FixFox warns before closing so you do not lose context accidentally.

## When to use a support package

Create a support package when:

- a repair fails more than once
- the issue smells like account, certificate, permissions, or policy trouble
- the device is managed and Windows blocks the local action you need
- you want to hand clean evidence to IT or to the FixFox seller

## When to stop and escalate

Use FixFox first for normal Windows cleanup, browser cleanup, network resets, meeting-device checks, and repair prep.

Escalate instead of forcing more fixes when:

- sign-in, certificate, MFA, or permissions errors are the real blocker
- the device is managed and policy keeps reversing the change
- Windows is unstable after updates, servicing repair, or reboot
- the issue points to destructive recovery, reinstall, or possible hardware failure

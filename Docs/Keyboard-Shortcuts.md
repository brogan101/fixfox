# Keyboard Shortcuts

## Global

- `Ctrl+K` or `Ctrl+Space`: Open global search
- `Ctrl+H`: Open Activity
- `Ctrl+F`: Open Repair Library
- `Ctrl+D`: Open Home
- `Ctrl+,`: Open Settings
- `Ctrl+B`: Open Support Package
- `Ctrl+Shift+R`: Run the last useful action again
- `F5`: Refresh the current page
- `F1`: Open help for the current page
- `Alt+Left`: Go back to the previous page
- `?`: Open the keyboard shortcuts reference

## Fix Center

- `/`: Focus the fix search box

## History

- `Ctrl+A`: Select all visible receipts
- `Ctrl+E`: Export the selected receipts

## Automation

- `Ctrl+N`: Focus the automation rule list so you can start editing quickly

## Toolbox

- `Enter`: Open the focused tool
